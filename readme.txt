# My GitHub Portfolio Project

Welcome to my GitHub Portfolio! This project showcases my work as a software engineer and provides a platform for me to share my skills, projects, and experiences with the world.

## Introduction

I'm John B. Freibaum, and I'm excited to present my GitHub Portfolio Project. This project is a personal website designed to highlight my passion for technology and innovation. Through this portfolio, I aim to demonstrate my expertise, showcase my projects, and connect with like-minded individuals and potential employers.

## Features

- Home Section: A welcoming introduction that provides an overview of who I am and what I do.
- Bio Section: A glimpse into my background, skills, and experiences as a software engineer. It includes a short background video.
- LinkedIn Section: A direct link to my LinkedIn profile, allowing visitors to connect with me professionally.
- Projects Section: A collection of my notable projects, each with buttons to access the source code and live demos. This section highlights my technical abilities and showcases my accomplishments.
- Resume Section: An option to view and download my professional resume, providing a comprehensive overview of my skills, education, and work history.

## Installation

As this is a GitHub portfolio project, no installation is required. Simply access the portfolio website by visiting the provided URL.

## Usage

Feel free to explore my portfolio website and navigate through the different sections. You can learn more about me, view my projects, and access my resume for a detailed overview of my qualifications. The provided links allow you to explore my projects in depth, visit my LinkedIn profile, or get in touch with me through email.

## Technologies Used

- HTML: Used for structuring the content and layout of the portfolio website.
- CSS: Applied for styling and visual enhancements.
- JavaScript: Implemented for interactivity and functionality, such as handling sidebar navigation and button clicks.
